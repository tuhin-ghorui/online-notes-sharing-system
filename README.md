# Online Notes Sharing System

A web-based application developed using PHP and MySQL where students can upload, view, and download academic notes.

## Features
- Student registration and login
- Upload notes (PDF, Image, DOC, PPT)
- View and download notes
- Subject-wise search
- Admin moderation
- CRUD operations

## Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP

## How to Run Locally
1. Install XAMPP
2. Place project inside htdocs
3. Import database.sql into phpMyAdmin
4. Run at http://localhost/ONLINE_NOTES_SHARING/

## Note
This project requires a PHP server and MySQL database.
